## 字符串内嵌函数
```
# 删除开头或结尾的指定字符
>>> name = '*xusx***'
>>> print(name.strip('*'))
xusx
>>> print(name.lstrip('*'))
xusx***
>>> print(name.rstrip('*'))
*xusx
```
```
# 将字符串的第一个字符串转换成大写
message = 'kubernetes nginx'
msg = message.capitalize()
print(msg)

>> Kubernetes nginx

```
```
# 每个单词的首字母大写
msg = message.title()
print(msg)

>> Kubernetes Nginx

# 判断首字母是否大写,返回的结果是布尔类型
result = msg.istitle()
print(result)

>> True

```
```
# 将全部字符串转换成大写
msg = message.upper()
print(msg)

>> KUBERNETES NGINX

# 判断所有字母是否大写,返回的结果是布尔类型
result = msg.isupper()
print(result)

>> True

```
```
# 将全部字符串转换成小写
msg = message.lower()
print(msg)

>> kubernetes nginx

# 判断所有字母是否小写,返回的结果是布尔类型
result = msg.islower()
print(result)

>> True

```

# format的三种玩法
>>> res = '{} {} {}'.format('xusx', 28, 'male')
>>> print(res)
xusx 28 male

>>> res = '{1} {0} {1}'.format('xusx', 28, 'male')
>>> print(res)
28 xusx 28

>>> res = '{name} {age} {sex}'.format(age=28, sex='male', name='xusx')
>>> print(res)
xusx 28 male
