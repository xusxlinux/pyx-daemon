## 字符串
```
# 删除开头或结尾的指定字符
>>> name = '*xusx***'
>>> print(name.strip('*'))
xusx
>>> print(name.lstrip('*'))
xusx***
>>> print(name.rstrip('*'))
*xusx


# upper,lower
>>> name = 'xusx'
>>> print(name.upper())
XUSX
>>> print(name.lower())
xusx


# startswith,endswith
>>> name = 'xusx_why'
>>> print(name.endswith('why'))
True
>>> print(name.startswith('xusx'))
True


# format的三种玩法
>>> res = '{} {} {}'.format('xusx', 28, 'male')
>>> print(res)
xusx 28 male

>>> res = '{1} {0} {1}'.format('xusx', 28, 'male')
>>> print(res)
28 xusx 28

>>> res = '{name} {age} {sex}'.format(age=28, sex='male', name='xusx')
>>> print(res)
xusx 28 male
```
