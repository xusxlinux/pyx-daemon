
#### 算数运算
| 运算符   | 描述           |  示例      |
| -------- | -----:         | :----:     |
| +        |两个对象相加    |a+b         |
| -        |两个对象相减    |a-b         |
| *        |两个对象相乘    |a*b         |
| /        |两个对象相除    |a/b         |
| %        |取余            |a%b         |
| **       |次幂            |a**b        |
| //       |返回商的整数部分|7//2 等于 3 |
