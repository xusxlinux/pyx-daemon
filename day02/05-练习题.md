#### 1. 使用while循环输出1 2 3 4 5 6     8 9 10
```
count = 0
while count < 10:
    count += 1
    if count == 7:
        continue
    print(count)
```
#### 2. 求1-100的所有数的和
```
sum = 0
count = 1
while count <= 100:
    sum += count
    count += 1
    print(sum)
```
#### 3. 输出 1-100 内的所有奇数
```
sum = 0
count = 1
while count <= 100:
    if count % 2 != 0:
        print(count)
    count += 1
```
#### 4. 输出 1-100 内的所有偶数
```
sum = 0
count = 1
while count <= 100:
    if count % 2 == 0:
        print(count)
    count += 1
```
#### 5. 求1-2+3-4+5 ... 99的所有数的和
```
res = 0
count = 1
while count <= 3:
    if count%2 == 0:
        res -= count
    else:
        res += count
    count += 1
print(count)
```
