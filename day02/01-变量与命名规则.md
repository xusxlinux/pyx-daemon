#### 命名规则
由数字、字母、下划线组成  
不能是纯数字，也不能以数字开头  
不能用关键字  
不能太长  
不要用中文  
要有意义  
区分大小写  
建议`驼峰`或者`下划线`

#### 头文件
```
#!/usr/bin/env python
# -*- coding: utf-8 -*- 
```
#### 小整数池
为避免因创建相同的值而重复申请内存空间所带来的效率问题，python解释器会在启动时创建小整数池,范围[-5,256],在这个范围内的小整数对象是全局解释器范围内被重复使用，永远不会被GC回收
```
>>> a=4
>>> id(a)
9331008

>>> b=3
>>> b+=1
>>> id(b)
9331008
```
