#### 格式化输出
```
name = input('name>>:')
age = input('age>>:')
print('my name is %s my age is %s' % (name, age))
```

```
name = input('name>>:')
age = input('age>>:')
sex = input('sex>>:')

msg = '''
-----info of %s-----
Name: %s
age: %s
sex: %s
---------end--------
''' % (name, name, age, sex)
print(msg)
```
