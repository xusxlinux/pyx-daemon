#### 格式化输出
`%s`字符串占位符：可以接收字符串，也可以接收数字

`%d`数字占位符：只能接收数字
```
name = input('name>>:')
age = input('age>>:')
print('my name is %s my age is %s' % (name, age))
```

```
name = input('name>>:')
age = input('age>>:')
sex = input('sex>>:')

msg = '''
-----info of %s-----
Name: %s
age: %s
sex: %s
---------end--------
''' % (name, name, age, sex)
print(msg)
```
